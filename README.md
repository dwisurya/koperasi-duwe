# koperasi-duwe

Aplikasi koperasi - RBAC admin panel berbasis Laravel 12.

## Fitur

- Manajemen Anggota, Simpanan, Pinjaman, Bunga Pinjaman
- RBAC (Role-Based Access Control) dengan Spatie Permission
- Multi bahasa (Indonesia/Inggris)
- Approval workflow pinjaman
- Cetak kontrak pinjaman
- Simulasi cicilan
- Dashboard statistik
- Responsive design (mobile/tablet/desktop)
